# Fatecita

Bem-vindo ao repositório "Fatecita"! Este repositório contém os códigos desenvolvidos em aula com a linguagem de programação Java. Aqui, trabalhamos conceitos fundamentais da programação orientada a objetos e aplicamos práticas modernas de desenvolvimento de software.

## Ementa

Este curso abrange os seguintes tópicos:

- **Conceitos de Orientação a Objetos:** Classes, Objetos, Encapsulamento, Herança e Polimorfismo.
- **Padrões de Projeto:** Introdução aos princípios e aplicação de padrões de projeto.
- **Declaração de Classes e Objetos:** Criação e manipulação de classes e objetos em Java.
- **Classe Abstrata e Métodos:** Uso de classes abstratas, métodos e sobrecarga de métodos.
- **Herança Múltipla:** Conceitos teóricos e suas implicações na programação.
- **Modificadores de Acesso e Construtores:** Utilização de modificadores de acesso e construção de objetos.
- **Manipulação de Exceções:** Compreensão e aplicação de tratamento de exceções.
- **Arquitetura em Camadas:** Aplicação de conceitos de arquitetura em camadas, incluindo o padrão MVC (Model-View-Controller).
- **Interface Gráfica:** Criação de interfaces gráficas de usuário (GUIs) em Java.
- **Teste de Software:** Métodos e práticas de teste para garantir a qualidade do software.

## Objetivo

O objetivo deste curso é capacitar os alunos a:

- Utilizar a linguagem Java para aplicar conceitos de orientação a objetos como abstração, encapsulamento, herança e polimorfismo.
- Compreender o relacionamento entre classes e programar tratamento de exceções.
- Criar interfaces gráficas interativas para usuários.
- Aplicar conceitos da arquitetura MVC para desenvolvimento de software.
- Conhecer e utilizar frameworks de desenvolvimento front-end e back-end.
- Aplicar práticas de versionamento e documentação de aplicações.

## Estrutura do Repositório

O repositório está organizado da seguinte maneira:

- **/src:** Contém o código-fonte dos projetos desenvolvidos.
- **/docs:** Documentação e recursos de apoio para as aulas.
- **/exercicios:** Exercícios e desafios práticos para fixação do conteúdo.
- **/projetos:** Projetos completos desenvolvidos ao longo do curso.
